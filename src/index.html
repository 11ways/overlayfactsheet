Overlay Fact Sheet

Introduction, Definition, and History of Web Accessibility Overlays

Summary: Overlays are a broad term for technologies aimed at improving the accessibility of a website by applying third-party source code (typically JavaScript) to make improvements to the front-end code of the website.

Strengths and Weaknesses of Overlay "Widgets"

Summary: Overlay widgets unnecessary and poorly placed in the technology stack.

Strengths and Weaknesses of Automated Repair

Summary: While some automated repair is possible, customers should be discouraged from using an overlay as a long-term solution.

Fitness for Achieving Compliance with Accessibility Standards

Summary: While the use of an overlay may improve compliance with a handful of provisions in major accessibility standards, full compliance cannot be achieved with an overlay.
